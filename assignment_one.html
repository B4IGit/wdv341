Research and define the following Git related terms

version control software: also known as source control, tracks and manages changes to code.

add: $ git add . - this command lets you know that the file is important. It adds the document before it is committed and pushed.

commit: $ git commit -m " " - "save button", this commits or saves the file. "-m" allows you to message, comment what you are committing.

push: with $ git push origin master, this sends the files to the remote repo, in my case GitHub.

pull:$ git pull - is used to download and integrate remote changes.

clone:$ git clone - makes a copy of an existing Git repo.
